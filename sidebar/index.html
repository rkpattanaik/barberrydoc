    How to set up sidebar?

    Set sidebar position

    1. In your Customize Theme screen

    2. Under the Theme settings tab, click General

    Add block for sidebar

    1. In your Customize Theme screen, select Home pages

    2. Under the Sections tab, click Sidebar

    ##

    A section allows you to add different types of blocks, Including block types:

    Blocks type Function
    News Letter Allows to add an email entry area.
    Tags Product Show all tags of a product of collection
    Collection Allows adding a collection
    Banner Add a banner, which allows you to adjust heading.
    Testimonials Allows you to enter and display information of a celebrity.
